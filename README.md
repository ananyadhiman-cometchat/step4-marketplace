# Marketplace — baseline (`main`)

A buy/sell marketplace across **web + Android + iOS + backend**. This is the **baseline** app with
**no CometChat** — a realistic, production-shaped starting point. The chat/voice/video integration
lives on [`feature/cometchat-integration`](https://github.com/ananyadhiman-cometchat/step4-marketplace/tree/feature/cometchat-integration).

## Stack
| Component | Tech |
|---|---|
| `web/` | Next.js (App Router) + Tailwind |
| `mobile/` | Expo / React Native |
| `backend/` | FastAPI + Postgres |

## Features (baseline)
- RBAC roles (admin / seller / buyer / moderator) with JWT auth.
- Listings: browse, detail, create (seller), moderate (admin/moderator).
- Seeded users + a sample listing ("Vintage Camera").
- Login smoke (web Playwright `@smoke`, mobile Maestro flow).

## REST contract (canonical, shared across STEP4 use cases)
- Base path `/api`; health `GET /api/health` → `{"status":"ok"}`.
- `POST /api/auth/login` `{email,password}` → `{token, user:{uid,name,role,avatar_url}}` (Bearer JWT).
- Lists are paginated + envelope-wrapped: `GET /api/<resource>?page&limit` →
  `{"data":[...], "pagination":{page,limit,total}}`. Detail returns the bare object.
- Errors: `{"detail":"..."}` with correct HTTP status. Money serializes as strings; timestamps ISO-8601.
